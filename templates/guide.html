Celestia 2D - DOCUMENTATION

Overview

Celestia 2D is a simulation of celestial objects including planets, stars, neutron stars and black holes. Users can create objects with specific mass, radius, and velocity, and the simulation will automatically classify them based on the density and size.
Key Features
  • Gravtiy based interactions (scaled at 5e-4 G )
  • Collision and orbital motion
  • Dynamic visual effects ( glow, ring, rotation )

    • How to Spawn Objects

    1. Click on the canvas to place the objects at the cursor location
    2. Input fields ( all optional, defaults used if blank )
    3. The simulation will automatically calculates density and assigns an object type

    Celestia Quick Reference Table

    Type Density Range Suggested Radius Notes
    Planet 0.2 – 5 10–30 Default planet
    Giant Planet 0.2 – 5 35+ Rings, visually large
    Star 5 – 20 8–20 Glowing
    Neutron Star 20 – 50 5–10 Very dense
    Black Hole 50+ 5–15 Rotates, strong gravity

    Tips for Users

  • Larger radius with same mass => lower density => may as classify as planets instead of star
  • High velocities can allow objects to escape gravity of nearby objects
  • Rrecommended rages
  • Mass: 100 - 100000000
  • Radius: 5 - 50
  • Experiment with the combinations to see interactions like collision, orbititng or clustering

    • Visual Effects

  • Black Holes: Rotating, glowing gradient, optional accretion disk
  • Stars & Neutron Stars: Glowing halo and layered gradients
  • Giant Planets: Rings, large radius
  • Planets: Subtle radial gradients and outlines

    • Final Notes

  • All units are simulation units, not real-world units
  • You can tweak density thresholds, color, and visual effects to custimise your univers
  • Encourage experimentation - Thats the fun part of Celestia 2D
