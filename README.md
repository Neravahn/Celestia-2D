CELESTIA 2D

Celestia is an interactive 2D space-physics simulator built with JavaScript + HTML Canvas.
You can create stars, planets, neutron stars, giant planets, and even black holes - all obeying real gravitational physics.
Objects orbit, collide, form trails, and can be saved/loaded as full simulations.
